# Blood Demand Prediction & Donor Management System

## About this project
Hospitals often run out of blood during emergencies because they cannot predict demand in advance.
This project solves that problem using Machine Learning.

I built a model that predicts blood demand based on hospital data helping hospitals manage
their blood inventory, reduce shortages, and save lives.

## What it does
- Predicts future blood demand using historical hospital data
- Tracks donor eligibility and donation history
- Monitors blood inventory levels in real time
- Alerts when stock is running low

## Tools & technologies used
- Python
- Pandas — data cleaning and manipulation
- Scikit-learn — machine learning model
- Matplotlib / Seaborn — data visualization
- Excel — data input and reporting

## How to run this project
1. Download all files from this repository
2. Install required libraries:
   pip install pandas scikit-learn matplotlib seaborn
3. Open and run the main Python file
